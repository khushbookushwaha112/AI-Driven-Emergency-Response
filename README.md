# 🚨 AI-Driven Emergency Response System

An AI-based system that analyzes crime-related text and predicts the type of emergency using Machine Learning.

---

## 📌 Project Overview

The **AI-Driven Emergency Response System** helps analyze crime or emergency descriptions and predicts the type of incident using a trained Machine Learning model.

This system can assist:

* 🚓 Police departments
* 🚑 Emergency response teams
* 🏥 Public safety organizations

to quickly understand incidents and take faster action.

---

## ⚙️ Technologies Used

* Python
* Flask
* Machine Learning (Scikit-learn)
* Natural Language Processing (TF-IDF)
* Pickle (.pkl) for model storage
* Render for deployment

---

## 📂 Project Structure

AI-Driven-Emergency-Response

* app.py → Main Flask application
* train_model.py → Model training script
* test_deployment.py → Deployment testing
* requirements.txt → Python dependencies
* Procfile → Deployment configuration

Model Files:

* crime_model.pkl → Trained ML model
* vectorizer.pkl → Text vectorizer
* crime_details.pkl → Crime category mapping

Dataset:

* data/ → Training dataset

## ▶️ How to Run the Project

### 1 Clone the Repository

git clone https://github.com/khushbookushwaha112/AI-Driven-Emergency-Response.git

### 2 Go to Project Folder

cd AI-Driven-Emergency-Response

### 3 Install Dependencies

pip install -r requirements.txt

### 4 Run the Application

python app.py
